TASK 1 — DATABASE

Add a table for magic link tokens in db.py.

SQL:

CREATE TABLE IF NOT EXISTS magic_link_tokens (
    id SERIAL PRIMARY KEY,
    email TEXT NOT NULL,
    token_hash TEXT NOT NULL,
    expires_at TIMESTAMP NOT NULL,
    used BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_magic_tokens_email
ON magic_link_tokens(email);

CREATE UNIQUE INDEX IF NOT EXISTS idx_magic_token_unique
ON magic_link_tokens(email, token_hash);

This table stores hashed login tokens.

Steps:

Accept email from request form.

Generate token:

Insert into DB:

email
token_hash
expires_at = NOW + MAGIC_LINK_EXPIRY_MINUTES
used = false

Send email via Resend API.

Endpoint:

POST https://api.resend.com/emails

Headers:

Authorization: Bearer RESEND_API_KEY
Content-Type: application/json

  "subject": "Your Clear Seeing login link",

}
