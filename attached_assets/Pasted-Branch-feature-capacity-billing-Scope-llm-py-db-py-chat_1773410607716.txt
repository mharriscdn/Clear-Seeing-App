TASK 3 — CAPACITY UNIT SYSTEM

Pricing for claude-sonnet-4-6:
  - Input: $3.00 per million tokens
  - Output: $15.00 per million tokens
  - Cache reads: $0.30 per million tokens

4x margin: user is charged 4x actual API cost.
1 capacity unit = $0.001 USD of actual API cost x 4.
Formula: capacity_units = ceil((actual_cost_dollars * 4) / 0.001)

Billing uses input_tokens, output_tokens, and cache_read_input_tokens only.
Do not include cache_creation_input_tokens in cost calculation.

In db.py, add deduct_capacity(user_id, input_tokens, output_tokens,
cached_tokens):
  - Calculate actual API cost from token counts using rates above
  - Apply 4x margin
  - Convert to capacity units, round up (integer)
  - Subtract from users.capacity_remaining
  - Return the integer capacity_units deducted
  - No floor enforcement yet — deduct and log only, no blocking mid-session
